## 🧩 Key Features

### Responsive Navigation
- **Desktop** – Horizontal menu with logo and links.
- **Mobile** – Hamburger menu that slides in from the right with smooth animation.
- **Sticky Navbar** – On scroll, the navbar gains a blurred background and compact padding.
- **Active Link Highlighting** – Current section link is highlighted automatically on scroll.

<br>

### Hero Section
- Full‑width hero with badge, title, subtitle, and two call‑to‑action buttons.
- Image placeholder with hover zoom effect.
- Stacked on mobile, side‑by‑side on tablet/desktop.

<br>

### Features / Services
- 6 service cards with icons, title, description, and “Learn More” link.
- Grid layout: 1 column (mobile) → 2 columns (tablet) → 3 columns (desktop).
- Hover effects: card lifts, icon background changes, link arrow animates.

<br>

### About Preview
- Two‑column layout: image (with floating badge) + text content.
- Stats section: Projects Done, Happy Clients, Team Members.
- Call‑to‑action button linking to contact section.

<br>

### Portfolio Preview
- Grid of project cards with image and overlay on hover.
- One card spans two rows on desktop for visual variety.
- Grid: 1 column (mobile) → 2 columns (tablet) → 3 columns (desktop).

<br>

### Contact CTA
- Gradient background with centered text, title, description, and two buttons.
- “Start a Project” opens email client; “Call Us” opens phone dialer.

<br>

### Footer
- Brand, description, social icons, and three link columns.
- Social icons change color on hover.

<br>

### Scroll Animations
- Elements with `.reveal` class fade in and slide up when they enter the viewport using IntersectionObserver.
- Smooth scrolling for all anchor links.

<br>

## 🧪 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Umaima-Manzoor/Nexus-Responsive-Layout.git
   cd Nexus-Responsive-Layout
   ```

2. **Open the project**
   - Double‑click `index.html` to open in your browser, or
   - Use a local development server (e.g., VS Code Live Server)

3. **No dependencies or build tools required** – it just works.
